Welcome to the Brew Bank!

*SUMMARY*
This app is a dynamic database used to collect coffee experiences. Users can add their favorite coffee shops and drinks to the collection, which others can use as a reference to enhance their own java journey.

*INSTALLATION INSTRUCTIONS*
This product does not have to be installed. It can be launched from a web browser with internet connectivity.

*HOW TO USE THIS APP*
Begin your journey on the home page and control your path with the navigation bar. It is recommended to first create a username and password by selecting the "CREATE ACCOUNT" option.

<img width="1078" alt="Screenshot 2022-03-14 at 12 55 34 PM" src="https://user-images.githubusercontent.com/95182715/158233722-2c8fb42f-4281-4b29-9672-e68f56d59b39.png">

Once you are established as a user, each page features a login/logout option in the upper right hand corner.

You will find a global collection of all brews added on the "VISIT BREW BANK" page. Each brew has the option for modification and deletion, making this a CRUD application.

<img>

To add a new location and drink to the Brew Bank, visit the "ADD NEW BREW" link. On this page you will be make additions to the Brew Bank by completing the form.

*TECHNOLOGIES USED*
This app was built as a full-stack project.  
-Express
-React
-JavaScript
-HTML
-CSS(Bootstrap) styling featuring grid and flex-box components
-Mongoose: It utilizes a MongoDB database to manage users and data.
-Canva.com: To create graphics.
